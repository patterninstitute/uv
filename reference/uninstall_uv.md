# Uninstall uv

Prints the steps required to remove uv from your system. This function
does not modify your system — removal is left to the user to perform
manually, since it involves deleting binaries and optionally clearing
stored data.

## Usage

``` r
uninstall_uv()
```

## Value

Invisibly returns `NULL`. Called for its side effects.

## See also

[`install_uv()`](https://www.pattern.institute/uv/reference/install_uv.md)

## Examples

``` r
if (FALSE) uninstall_uv() # \dontrun{}
```
